# Calculadora

- Uma calculadora escrita em elixir para estudos

## Como usar?

- Rode um ```git clone https://github.com/ArthurDeveloper/calculator-elixir.git```
- Rode ```iex -S mix```
- No terminal interativo, digite ```Calculator.main(%{value1: 5, signal: "+", value2: 5})```. Voce pode trocar o sinal por + (soma), - (subtracao), * (multiplicacao), ** (potencia), v (raiz quadrada) e / (divisao). E os numeros tambem podem ser alterados, lembre de sempre passar o sinal entre aspas e o numero sem aspas
